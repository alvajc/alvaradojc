Book Recycle

Developed February 2016

This is a fictional site that allows users to buy and sell used books. When a user is entering their information to sell their books the form will show input validation as the user types. The search used books input box is indexed. If a user searches "harry", they will see all the Harry Potter books that are in the MongoDB database.

Technologies Used:

Javascript, HTML5, CSS3, NodeJS, ExpressJS, MongoDB, AngularJS, and jQUery. Bootstrap for responsive design. App is deployed on Heroku with database hosted on mLab.

foodGRADES

Developed January 2016

Users can search their favorite restaurant and view the most current L.A. County Dept. of Health inspection grade. They will also see yelp reviews and a google map with restaurant location. Health Dept. API is not fully reliable with search results but if a user searches for "Starbucks" in "Cerritos" they will see correct information.

Technologies Used:

Javascript, HTML5, CSS3, NodeJS, ExpressJS, API's from Yelp, Google Maps and Los Angeles Department of Health. Bootstrap for responsive design. App is deployed on Heroku.
